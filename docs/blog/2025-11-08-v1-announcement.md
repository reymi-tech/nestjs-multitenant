# Announcing nestjs-multitenant v1.0.0

We're excited to announce the first stable release of **nestjs-multitenant**, a comprehensive multi-tenant solution for NestJS applications!

<!--truncate-->

## What is nestjs-multitenant?

nestjs-multitenant is a production-ready package that brings schema-per-tenant multi-tenancy to your NestJS applications using PostgreSQL.

## Key Features

- Schema-per-Tenant Architecture
- Multiple Tenant Resolution Strategies
- Dynamic Connection Pooling
- Type-Safe Decorators
- Built-in Admin Module
- Flexible Entity Registry
